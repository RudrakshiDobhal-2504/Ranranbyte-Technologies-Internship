# Guard Logic Challenge (C++)

This project implements a simple guard-based decision system for selecting a safe door among three options.

## Problem
There are three doors:
- One leads to a safe exit
- One leads to a trap
- One leads back to the starting point

The system decides the correct door based on logical conditions.

## Approach
- Validated input to ensure correct door types (S, T, R)
- Ensured exactly one safe door for logical consistency
- Used modular functions for clarity and readability
- Applied conditions like time and emergency in decision-making

## Features
- Input validation with retry
- Clear door status display
- Structured and readable C++ code
- Condition-based decision logic
